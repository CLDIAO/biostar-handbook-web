These are pages that may contain plain html, markdown (see below) and can operate as django templates.

The django templates can access data loaded from context.py

For example context.say_hello will be rendered as: {{ context.say_hello }}

Restart the server if you alter context.py or if you add a new file. Pyblue will reload changes in existing files but won't pick up new files or a changed context.py

Or it can read the context.numbers in a for loop here: {% for value in context.numbers %} {{ value }} {% endfor %}
